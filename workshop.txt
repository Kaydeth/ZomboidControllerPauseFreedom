title=Controller Pause Freedom
description=The goal of this mod is to allow a controller player to pause time and manipulate the UI with the same freedom that a mouse an keyboard player is afforded. Removes some restrictions on when a player can pause the game with a controller and allows you to navigate menus when paused.
description=
description=Features:
description=* Allows player to pause time when a menu is open. Such as Invetory, Craft, Build, or Health menu.
description=* Allows player to pause time while moving a selection cursor. Such as when trying to pick up furniture.
description=* Allows player to open and close menus in the back button wheel while the game speed is paused.
description=* The pause button will still open the main menu when no menus are currently open.
description=
